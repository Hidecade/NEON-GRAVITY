# NEON GRAVITY: ORBITAL

宇宙を幾何学的な死の世界へ変換する謎の機械生命体『アキシオム』。
空間を歪ませる「ネオン・グリッド」の最深部へ向け、次元潜行型特務戦闘機「エメラルド・フェニックス」が発進する——。

**NEON GRAVITY: ORBITAL** は、重力と幾何学が交差する、高密度な全方位シューティングゲームです。
空間を歪ませるワームホール、ネオンに輝くベクターグラフィックス、そして攻防一体の「サテライト・システム」を駆使して、無限に湧き出る機械生命体を殲滅してください。

## 【プロローグ：宇宙の静寂】
西暦23XX年。人類は銀河の半分の開拓を終えていたが、ある日境界宙域の星々が次々と「消失」していることに気づく。
破壊されたのではない。星も、デブリも、生命も、すべてが**「完璧な幾何学的形状」へと再構築され、生命活動を停止**していたのだ。

それは『アキシオム（公理）』と呼ばれる、起源不明の自律型宇宙開拓マシンの群れだった。彼らの目的は、宇宙のあらゆる混沌（有機物）を排除し、完全な数式に基づく「究極の秩序」へ変換すること。
彼らが侵攻した宙域は、重力場が異常なねじれを起こし、空間そのものが発光する**『重力歪曲宙域（ネオン・グリッド）』**と化してしまう。通常の地球防衛軍の艦隊は、このグリッドに触れた瞬間に原子レベルで分解され、全滅した。

## 【希望：エメラルド・フェニックス】
アキシオムの侵攻が地球圏の絶対防衛線に迫る中、極秘裏に開発されていた一機の試作戦闘機が発進する。
**次元潜行型特務戦闘機「エメラルド・フェニックス」**。

この機体の最大の特徴は、敵と同じ重力歪曲機関を搭載していること。アキシオムの残骸から放出されるエネルギー結晶（クリスタル）を吸収し、戦闘中に自らの武装（WEAPON LEVEL）をリアルタイムで有機的に進化させる能力を持つ。
搭乗するのは、グリッドの重力負荷に耐えられるよう、神経系を機体と直結（オービタル・リンク）させた「あなた（COMMANDER）」ただ一人。

目標はただ一つ。敵群の最深部、宇宙を幾何学空間に書き換えている巨大な発生源を破壊すること。

## 【ACT 1：ネオン・グリッドへのダイブ（Stage 1〜4）】
空間が緑や青に光る網目（グリッド）として視覚化される異次元空間への突入。
迎撃に現れたのは、無機質に群れをなす紫の『トライアングル』や、機械的な尾を引きながら高速で流動する白の『タッドポール』。そして、赤く長い体節をうねらせて喰いついてくる中ボス『ドラゴン』。
あなたは敵を破壊し、こぼれ落ちるクリスタルを拾い集めながら、機体の主砲を広角化し、無敵のエネルギーバリアや極太の収束レーザーを解放して突き進む。

## 【ACT 2：進化する幾何学（Stage 5〜8）】
敵の中枢に近づくにつれ、アキシオム側も「予測不能な生命体」であるあなたを危険因子と認識し始める。
空間を引き裂いて現れる『ワームホール』の数が増加。自転しながら正確にあなたの未来位置をレーザーサイトで狙い撃つオレンジ色の『ハンター』や、破壊しても分裂して襲い掛かる『アステロイド（星の残骸）』が道を阻む。

各宙域を統括するボスたちは、段階を踏んで複雑化していく。
『TRI-FORTRESS（正三角形）』から始まり、『HEXAGON-NEST（正六角形）』、『OCTAGON-COMMAND（正八角形）』へと、その多角形の頂点が増えるごとに、放たれる幾何学的な弾幕とレーザーは苛烈さを増していく。

## 【ACT 3：絶対防衛線・ボスラッシュ（Stage 9）】
敵本星の眼前に広がる最終防衛ゲート。
ここでは空間の法則が乱れ、これまで破壊してきた歴代の多角形ガーディアンたちが、より凶悪なエネルギーを帯びて次々と再構築（リスポーン）される。
弾幕の嵐、息をつく暇もない連戦。あなたは研ぎ澄まされた反射神経と、蓄積したサテライト兵器（ホーミングミサイル）を駆使し、この幾何学の悪夢を単機で突破する。

## 【CLIMAX：創世方舟ジェネシス・アーク（Stage 10）】
ゲートを抜けた先、虚無の空間に鎮座していたのは、12の頂点を持つ規格外の超巨大要塞**『GENESIS-ARK（創世方舟）』**。
それはただの戦艦ではなく、この宇宙を「再計算」して作り直している巨大な演算コアだった。

全方位へのレーザー放射、空間を削り取るグラビティ・ハッチ、無数のホーミングミサイル、そして巨大な質量による突進。ダメージを与えるたびに真っ赤に発光し、激しい火花と白煙を吹き出しながら、ジェネシス・アークは狂ったように迎撃してくる。

激闘の末、あなたの放った渾身のレーザーがアークの中心で脈打つ真紅のコアを撃ち抜く。
時間が止まったかのようなスローモーションの中、アークの巨体が内部から連鎖的に崩壊。断末魔のような重力波を放ち、周囲のアステロイドや残存艦隊をすべて巻き込んで、眩い純白の光へと消滅していく。

## 【エピローグ：MISSION COMPLETE】
発光していたネオン・グリッドが静かに消え去り、元の美しい漆黒の宇宙と、自然な星々の瞬きが戻ってくる。
画面に浮かび上がる「ALL SECTORS LIBERATED（全宙域解放）」の文字。
傷つきながらも緑色の推進光（ジェット）を輝かせるエメラルド・フェニックスは、孤独な軌道（ORBITAL）を描きながら、静かに帰還の途につく。

あなたのスコアは、宇宙を救った英雄の記録として「WORLD RANKING」の歴史に永遠に刻まれるのだった。

## 🎮 ゲームの特徴 (Features)

* **Twin-Stick Action:** 移動と射撃を独立して操作する、直感的なツインスティック・スタイル。PC（キーボード＆マウス）とスマートフォンの両方に対応。
* **Gravity & Grid Physics:** 爆発やワームホールの重力干渉により、背景のグリッド空間がリアルタイムに美しく歪みます。
* **Satellite System:** 敵の残骸（クリスタル）を回収すると自機の周囲に「サテライト」が展開されます。
    * **防御:** 敵弾を防ぐ「盾」として機能します。
    * **攻撃:** 一斉発射（Launch）することで、強力な「ホーミングミサイル」として敵を追尾します。
* **Dynamic Visuals:**
    * ヒット時の火花、撃破時の派手なパーティクル拡散、壁面への着弾エフェクトなど、爽快感を重視したビジュアルフィードバック。
    * 自機のパワーアップに伴い、攻撃エフェクトが派手に進化（最大レベルで「エメラルド・フェニックス」のエフェクトが発動）。
* **Global Ranking:** Firebaseを使用したリアルタイム・オンラインランキング機能を搭載。世界中のプレイヤーとスコアを競えます。

## 🕹️ 操作方法 (Controls)

### スマートフォン / タブレット (Mobile)
画面上に表示されるバーチャルコントローラーを使用します。

* **左スティック:** 移動 (Move)
* **右スティック:** 照準 (Aim) & 自動射撃
* **LAUNCHボタン:** サテライト一斉発射 (Launch Satellites)

### パソコン (PC)

| Action | Key | Note |
| :--- | :--- | :--- |
| **移動 (Move)** | 矢印キー (`↑` `↓` `←` `→`) | |
| **射撃 (Shoot)** | スペースキー (`Space`) | 押しっぱなしで連射 |
| **サテライト発射** | `Z` または `X` | 敵を追尾するミサイルとして射出 |

## 👾 敵キャラクター (Enemies)

* **TRIANGLE:** 基本的な敵。編隊を組んで出現し、プレイヤーを取り囲みます。
* **CUBE:** アイテムキャリア。破壊すると必ずパワーアップアイテムをドロップします。
* **TADPOLE:** 素早い動きで接近してくる小型機。尾を引くエフェクトが特徴。
* **DRAGON (Armored Train):** 複数のコンテナ装甲が連結された巨大な機械竜。体をくねらせて画面を制圧します。
* **HUNTER:** 高速で接近し、弾をばら撒いて離脱するヒット＆アウェイ戦法をとる危険な敵。
* **ASTEROID:** 宇宙を漂う岩石。出現直後は無害ですが、**一定時間経過すると赤く変色して「覚醒」し、猛スピードでプレイヤーを追尾**し始めます。
* **BOSS (Geometric Fortress):** 各ステージの最後に待ち受ける巨大要塞。多角形の形状を持ち、複数の砲台から弾幕を展開します。

## 💎 パワーアップ & アイテム (Items)

緑色の敵（CUBE）や編隊を倒すと出現します。

* **[W] Weapon (Level Up):** 通常ショットが強化されます（最大Lv7）。拡散範囲が広がり、最終段階では特殊エフェクトを纏います。
* **[L] Laser:** 一定時間、敵や障害物を貫通する強力なレーザーを発射します。
* **[I] Invincible:** 一定時間、無敵状態になります。体当たりで敵を粉砕可能です。
* **[S] Shield:** シールド（HP）を回復します。
* **Crystal (緑のひし形):** サテライトを1つ補充します（最大12個）。

## 🚀 攻略のヒント (Tips)

1.  **サテライトの運用:** 常にクリスタルを回収してサテライトを維持しましょう。ボスなどの硬い敵には、サテライトを一斉発射（Launch）して大ダメージを与えるのが有効です。
2.  **アステロイドの処理:** 白い岩（Asteroid）は放置すると赤くなり、非常に危険な追尾弾となります。分裂した後も油断せず、早めに処理しましょう。
3.  **敵数制限とワームホール:** 画面内の敵が多すぎると新しい敵（ワームホール）が出現しません。処理落ちを防ぐためですが、逆に言えば敵を倒さない限り次へ進めないため、積極的な撃破が必要です。
4.  **硬い敵への対処:** ボスやドラゴンの装甲に弾が弾かれる（カキンという音と火花）場合でも、少しずつダメージは蓄積しています。諦めずに撃ち込みましょう。

## 🛠️ 技術スタック (Tech Stack)

* **HTML5 Canvas API:** 描画エンジン（加算合成、パーティクルシステム、多関節アニメーション）
* **Web Audio API:** 効果音（SE）のリアルタイムシンセサイズ生成（外部音声ファイル不使用により超軽量化を実現）
* **Firebase (Firestore):** リアルタイム・オンラインランキングシステム
* **Vanilla JavaScript:** ライブラリやフレームワークを使用しない、ピュアなJavaScriptによる実装。
